Les balises de base

<!DOCTYPE html>Definie le language

<html lang="en">Definie la langue, en(anglais), fr(francais)...

<head>Tête de la page, tout les parrametre de la page sont la comme le css ou le titre

    <meta charset="UTF-8">Encodage de la page pour que les caractere speciaux sois accepter

    <meta name="viewport" content="width=device-width, initial-scale=1.0">Pour que le responsive se fasse bien

    <title>Document</title>Met le titre de la page sur l'onglet

</head>Fin du head

<body>La où nous mettons les élements de notre page

</body>Fin du body

</html>Fin du html
