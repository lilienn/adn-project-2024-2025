Attribut a mettre dans le HTML

<p class="text">Je suis un paragraphe avec class="text"</p>

<p id="text2">Je suis un paragraphe avec id="text2"</p>

Sélecteur CSS

nom balise{} : Selectionne avec le nom de la balise HTML

.XXX{} : Selectionne les élement HTML avec la class

#XXX{} : Selectionne les élement HTML avec l'id

Propriété de base

color : nom de a couleur ou code hexa /ici rouge

font-size : nombre rem / ici 1.3rem

background : couleur ou url(lien image) /ici vert et couleur blanc
